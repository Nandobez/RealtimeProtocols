Realtime Protocols — head-to-head

Same data source, four transports. Each panel keeps a small log of the messages it receives and a metric strip showing latency, bytes and reconnects.

Server-Sent Events

transport: HTTP/1.1 stream direction: server → client

Long Polling

transport: HTTP/1.1 req/res direction: server → client (one shot per req)

WebSocket

transport: HTTP Upgrade → TCP frame direction: full duplex

WebTransport (QUIC / HTTP/3)

transport: UDP/QUIC stream direction: full duplex, multiplexed

Single Node.js process serves all four endpoints from the same ticker. Read the source.
